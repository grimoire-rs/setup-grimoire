grim — the AI-config package manager

Install on macOS / Linux:

curl --proto '=https' --tlsv1.2 -LsSf https://setup.grimoire.rs/sh | sh

Install on Windows (PowerShell 7.4+):

irm https://setup.grimoire.rs/ps1 | iex

In GitHub Actions:

- uses: grimoire-rs/setup-grimoire@v1

Installers are generated by cargo-dist and verify SHA-256 checksums. Docs: grimoire.rs · Source: grimoire-rs/grimoire
